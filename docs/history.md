## 2026-04-29

### 거래 차트 이미지 첨부
- 거래 상세에서 차트 이미지를 드래그·클릭·Ctrl+V로 업로드 가능
- 썸네일 클릭 시 라이트박스(원본 크기) 표시
- 이미지는 서버 `data/images/`에 파일로 저장, DB에는 파일명만 기록
- Trade 삭제 시 TradeImage 레코드 cascade 삭제
- 관련: `TradeImageZone`, `api/uploads`, `api/images`, `api/trade-images`

### 종목 마스터
- `/stock-master` 페이지에서 종목명 + 종목코드 직접 등록·수정·삭제
- 거래 저장/수정 시 마스터에서 symbolCode 자동 조회·세팅
- 한국투자증권 파싱처럼 symbolCode가 넘어오면 마스터에 자동 등록
- 마스터 등록/수정 시 같은 종목명의 기존 거래들 symbolCode 일괄 업데이트

### 네이버 금융 링크 개선
- symbolCode 있으면 종목 직접 링크, 없으면 검색 URL로 연결
- 검색 URL 형식 변경: `search/search.naver?query=...&endUrl=&encoding=UTF-8`
